    Automated Tests¶ ↑

    We ran automated test cases using Ruby on Rails’ own testing tool. To run it yourself, make sure you have Ruby, Rails, and Postgresql. If you don’t have the libraries required (i.e., if rails server throws errors) run: bundle install Then make sure you create the database so that it can run the test suite on the database: rake db:create db:migrate Finally, run the test suite: rails test If you have errors, feel free to ask Ben (besh3382@colorado.edu) and he can show it to you on his machine. Postgres is very picky when it comes to users. :)
